# GiteeOAuth2
OAuth2.0 登录验证Gitee&amp/Wechat示例

本项目实现了OAuth的后台请求处理功能
-  获取token
-  刷新token
-  获取用户数据


本项目code由前端获取，返回给后端gitee/auth接口

Gitee创建第三方应用：
> https://gitee.com/oauth/applications
更多细节请参见Gitee官方
> Gitee OAuth2.0 接口文档
> https://gitee.com/api/v5/swagger#/
> OAuth教程文档
> https://gitee.com/api/v5/oauth_doc#/
